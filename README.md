# YottaChain
Practical Project for the Blockchain Dev Camp at SoftUni

## Project Description
YottaChain is a **fully-functional blockchain** network consisting of **Nodes**, **Mining** software, **Wallet** software, **Faucet** app, **Block Explorer** app.
